### 题目描述

有一个盒子,盒子有两种数字,``0``和``1``每种数字有无限个,
现在有n( ``1<= n <= 20``)个人取数字,这n个人排成一排,先第一个人拿,然后依次去取.

问:一共有多少人可能性.按字典序从大到小输出所有的可能性

### 输入样例

```plaintext
3
```

### 输出样例

```plaintext
000
001
010
011
100
101
110
111
```
每个位置有两种可能性,0,1,所以最终有``2^2``行结果
