### 题目描述


斐波那契数列$(Fibonacci \; sequence)$是一个经典的数学问题，其定义如下：第一个和第二个数字都是$1$，接下来的每个数字都是前两个数字之和。例如，斐波那契数列的前几个数字是$1, 1, 2, 3, 5, 8, 13, 21, \cdots$,请编写一个程序，输出斐波那契数列中的第$n$个数字。

### 输入

一个整数$n$，表示要输出斐波那契数列的第$n$个数字$(1 \leqslant N \leqslant 100)$。

### 输出

斐波那契数列的第$n$个数字

### 示例

输入：

```
5
```
输出：

```
5
```
