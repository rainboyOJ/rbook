# Trie

Trie(字典树)是用来对字符串进行快速查找的一种的多叉树形结构.

## 操作

- 初始化,init
- 插入,insert
- 查找,search

注意

- 结点不是字符,边才是字符

## 性质

- 两个点的lca,表示两个字符串的最长公共前缀

## 功能

- 查找字符串,基本功能
- AC自动机,可以理解为树上KMP,后面写
- 维护异或极值,边插入边询问
- 维护异或和
- 01-trie 合并
- 可持久化字典
