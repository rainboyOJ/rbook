manimce 生成的动画的代码

## 使用

```bash
source ./venv/bin/active
```

## 说明

Q: 为什么选择manimce,

- [svg](https://github.com/zjl9959/algviz) 生成的动画有问题,不够丰富,有一些变化不能展示

## 动画列表

- 排序
- 动态规划
  - lis
    - 线性的动画
  - lcs
    - 暴力生成子序列集合
    - 矩阵生成
- 图论
